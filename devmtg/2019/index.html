About

The Embulk developers' meeting is a gathering of Embulk core developers and plugin developers :

  • To share the roadmap and planned changes in the Embulk core
  • To share demands to the Embulk core from plugins' viewpoints
  • To make consensus on the future plans

The meeting has been held invitation-based irregularly when important topics occured, so far as of 2019.

Topics

  • The pain of Embulk core/plugin development
    • Java dependencies embedded in the Embulk core
    • JRuby
    • Unclear interface defined between the core and plugins
  • The roadmap of the core development to struggle with the pain
    • Compatibility policy
    • Versioning strategy
    • Plugin catch-up strategy
    • Toward Embulk v1.0
  • How to build Maven-based Java plugins
  • DI implementation with Guice
  • Wants in the new SPI
    • What to implement by v1.0?
    • What to implement after v1.0 toward v2.0?
  • More to come?
